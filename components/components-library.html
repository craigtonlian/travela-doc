components

components library

This library contains the different reusable components that are used throughout the app.

This library should not be private or be placed in lib, as it is referenced throughout the app.

Classes

ActionableNavbar
ActionButton
Creates a floating action button that does something when clicked.
AirportCard
Card representation of an Airport object. Card shows important information about the airport, such as the name of the airport and the approximate distance from the user's current location.
ClickableIcon
Generates a generic icon button that does something when clicked.
Confirmation
Abstract class that generate Confirmation Dialogs which prompt users for their acknowledgement to events that occur within the app.
DocumentCard
Card representation of a Document object. This card also allow users to interact with the Document object and view the data attached to the object.
ElevatedIconButton
Creates a general full-width button with an icon that does something when clicked.
EmptyCard
Card that represents an empty collection.
FolderCard
Card representation of a Folder object. This card also allow users to interact with the Folder object and view the data attached to the object.
GeneralElevatedButton
Creates a general full-width button that does something when clicked.
GeneralNavBar
Abstract class that contains general nav bars that merely show the back button and the title.
GenericCard
Highest level Card class that grants all subclasses access to a random color generator and a random image generator.
HomeTripCard
Alternative Card representation of a Trip object. This Card widget is used exclusively on the Home Page of the app only.
HotelCard
Card representation of a Hotel object. Card shows information about the hotel, such as the name of the hotel and approximate distance from the user's current location.
PlaceOfInterestCard
Card representation of a PlaceOfInterest object. Card shows important information about the place of interest (POI), such as the name of the POI and the approximate distance from the user's current location.
Search
Reusable Search Bar.
SignInButtonBuilder
Builder class for the Sign In with ... buttons.
Skeletons
Abstract class that holds the different skeleton loaders used throughout the app.
Tags
Abstract class made to contain the different types of Tags used in the app.
TripCard
Card representation of a Trip object. This card also allow users to interact with the Trip object and view the data attached to the object.
WeatherCard
Card that represents the weather at a particular Trip destination, represented by a Location object.
