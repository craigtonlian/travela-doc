controllers

controllers library

This library contains the different Getx Controllers that are used throughout the app.

This library should not be private or be placed in lib, as it is referenced throughout the app.

Classes

ButtonScrollController
Simple class that helps control the hiding and showing of buttons on user scroll.
DocumentController
GetxController that controls Document objects, and provide interface methods to interact with them.
ExpenseController
GetxController that controls Expense objects, and provide interface methods to interact with them.
FilePickerController
GetxController that manages the state of the FilePicker Widget.
FolderController
GetxController that controls Folder objects, and provide interface methods to interact with them.
ItineraryController
GetxController that controls Itinerary objects, and provide interface methods to interact with them.
NearbyAirportController
GetxController to manage Airport objects, which represents Airports that are near the destination location of Trip objects.
NearbyHotelController
GetxController to manage Hotel objects, which represents Hotels that are near the destination location of Trip objects.
NearbyPlaceOfInterestController
GetxController to manage PlaceOfInterest objects, which represents places of interest that are near the destination location of Trip objects.
SplashController
GetxController for the Splash Screen.
TravellerController
GetxController to manage fellow Users who are collaborating on a Trip.
TripController
GetxController that controls Trip obejcts, and provide interface methods to interact with them.
UserController
GetxController that controls User objects, and provide interface methods to interact with them.
WeatherController
GetxController that manages the querying and parsing of weather data from the OpenWeather API.
