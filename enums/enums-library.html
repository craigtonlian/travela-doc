enums library

This library contains the different enums used throughout the application.

This library should not be private or be placed in lib, as it is referenced throughout the app.

Enums

DocumentCardEnum
Simple Enum thatp helps to distinguish the activities that users can do on any Document Card.
EmptyCardEnum
Simple enum used by the EmptyCard Widget to determine the string displayed to the user to alert them that the appropriate collection is empty.
SearchEnum
Enum that contains the different types of search results that can be queried with a Search widget.
