Public factory method that constructs a valid Document instance from another valid Document instance.

Document ID cannot be changed through this method. User ID can, however, since User ID represents the owner of the document, which can be changed.

The logic for changing the owner of the document would require further refining outside of this method, however the capability to change owners is provided here nonetheless.

Parameters:

  • document - The Document of interest
  • documentId - The Document ID
  • userId - The User ID of the owner of the Document
  • name - The name of the Document
  • color - The color of the tile used to display the tile
  • isStarred - The Starred status of the Document
  • storageUrl - The Document's cloud storage URL/URI
  • usersList - The list of Users who have access to this Document
  • lastUpdated - A running tally of the last time the Document was updated
