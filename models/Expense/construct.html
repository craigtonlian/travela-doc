Factory method that creates an instance of Expense.

Creation date defaults to the current time the Expense instance was created.

Parameters:

  • expenseId - The Expense ID of this Expense
  • userId - The User ID of the owner of this Expense
  • tripId - The Trip ID of which this Expense belongs to
  • description - The description of this Expense entry
  • currency - The currency of this Expense entry
  • value - The value of this Expense entry
  • entryDate - The Date of entry of the Expense
  • lastUpdated - The last time this Expense was updated
