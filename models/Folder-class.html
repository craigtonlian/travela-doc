Folder

Folder class

Represents a Folder object within the database.

Attributes:

  • folderId - The Folder ID
  • userId - The User ID
  • name - The name of the Folder
  • isStarred - The Starred status of the Folder
  • color - The color of the tile used to display the tile
  • storageUrl - The Folder's cloud storage URL/URI
  • usersList - The list of Users who have access to this Folder
  • foldersList - The list of Folders nested within this Folder
  • documentsList - The list of Documents nested within this Folder
  • creationDate - The date of creation of the Folder
  • lastUpdated - A running tally of the last time the Folder was updated
