Factory method that creates an instance of Itinerary.

Creation date defaults to the current time the Itinerary instance was created.

Parameters:

  • itineraryId - The Itinerary ID of this Itinerary
  • userId - The User ID of the owner of this Itinerary
  • tripId - The Trip ID of which this Itinerary belongs to
  • activity - String describing the activity planned out in the Itinerary Entry
  • location - The Location tagged to the Itinerary entry
  • startDate - The starting date of the Itinerary entry
  • endDate - The ending date of the Itinerary entry
  • documentsList - The list of documents attached to this Itinerary
  • lastUpdated - The last time this Itinerary was updated
