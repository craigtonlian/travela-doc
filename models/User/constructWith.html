Public factory method that constructs a valid User instance from another valid User instance.

User ID cannot be changed through this method.

Parameters:

  • userId - The User ID
  • firstName - The first name of the User
  • lastName - The last name of the User
  • displayName - The User's preferred display name
  • photoUrl - The User's profile picture URL
  • email - The User's email address
  • usersList - The User's list of Users who are friends
  • requestsList - The User's list of friends requests
  • tripsList - The User's list of Trips
  • documentsList - The User's list of Documents
