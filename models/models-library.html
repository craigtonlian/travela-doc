models

models library

This library contains the different Model classes that are used throughout the app.

This library should not be private or be placed in lib, as it is referenced throughout the app.

Classes

Airport
Class that holds information of an Airport.
Collections
A utility class that is used to designate a class as a Collection.
Colorable
An interface used to denote a Data class as capable of having colors.
ContainerCollections
Designates a class as a Collection that can contain other collections.
Document
Represents a Document object within the database.
Expense
Represents an Expense object within the database.
Folder
Represents a Folder object within the database.
Hotel
Class that holds information of a Hotel.
Itinerary
Represents a Itinerary object within the database.
Location
Class that help parse and store a location query from Google Places API.
LocationAutoComplete
Class used to contain Autocomplete data from Location searches.
PlaceOfInterest
Class that holds information of a Place of Interest.
SplashInfo
Class that is used to hold information used on the Splash Screen pages.
Trip
Represents a Trip object within the database.
User
Represents a User object within the database.
