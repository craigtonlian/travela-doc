services

services library

This library contains the different Getx Services that are used throughout the app.

This library should not be private or be placed in lib, as it is referenced throughout the app.

Classes

AuthenticationService
GetxService that contains the functions required to trigger an authentication event.
CurrentLocationService
GetxService that manages the current location of the User.
DatabaseService
GetxService that handles connections and queries between the server and the endpoints.
GooglePlacesService
GetxService that provides access to Google Maps API.
QueryService
GetxService that manages search queries within the app.
RequestService
GetxService that handles the friend requests made by users of the app.
WeatherService
GetxService that handles the querying of data from the OpenWeather API.
